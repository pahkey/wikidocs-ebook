API 토큰을 준비했다면 사용하는 LLM 도구에 위키독스 MCP 서버를 등록합니다.  
도구마다 설정 화면이나 명령은 조금씩 다르지만, 핵심은 MCP 서버 주소와 인증 헤더를 함께 등록하는 것입니다.

[TOC]

## 등록에 필요한 값

위키독스 MCP 서버를 등록할 때 필요한 값은 다음 두 가지입니다.

```text
MCP 서버 URL: https://wikidocs.net/mcp
인증 헤더: Authorization: Token 발급받은토큰
```

`발급받은토큰` 자리에는 앞 장에서 복사한 API 토큰을 넣습니다.  


## Claude Code에서 등록하기

Claude Code를 사용한다면 터미널에서 MCP 서버를 등록할 수 있습니다.  
기본 형태는 다음과 같습니다.

```bash
claude mcp add --transport http wikidocs https://wikidocs.net/mcp \
  --header "Authorization: Token 발급받은토큰"
```

등록 후에는 목록 명령으로 연결 상태를 확인합니다.

```bash
claude mcp list
```

목록에 `wikidocs`가 표시되고 연결 상태가 정상으로 나오면 사용할 준비가 된 것입니다.  
연결에 실패한다면 토큰 값과 헤더 형식을 먼저 확인합니다.

## Codex에서 등록하기

Codex CLI를 사용하는 경우에는 위키독스 MCP 서버를 등록한 뒤 설정 파일에서 인증 헤더를 연결합니다.

```bash
codex mcp add wikidocs --url https://wikidocs.net/mcp
```

그다음 Codex 설정 파일에서 위키독스 MCP 서버 항목을 확인합니다.  
인증 값을 직접 파일에 적기보다 환경 변수로 분리하면 토큰을 더 안전하게 관리할 수 있습니다.

```toml
[mcp_servers.wikidocs]
url = "https://wikidocs.net/mcp"
env_http_headers = { Authorization = "WIKIDOCS_AUTHORIZATION" }
```

환경 변수에는 다음과 같은 형태로 토큰을 넣습니다.

```bash
export WIKIDOCS_AUTHORIZATION='Token 발급받은토큰'
```

설정 후에는 연결 목록을 확인합니다.

```bash
codex mcp list
```

`wikidocs` 서버가 활성화된 상태로 표시되면 등록이 완료된 것입니다.

## Claude Desktop에서 등록하기

Claude Desktop처럼 설정 파일을 사용하는 도구에서는 MCP 서버 설정에 위키독스 항목을 추가합니다.  
설정 파일에는 서버 주소와 인증 헤더가 들어가야 합니다.

```json
{
  "mcpServers": {
    "wikidocs": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://wikidocs.net/mcp",
        "--header",
        "Authorization: Token 발급받은토큰"
      ]
    }
  }
}
```

설정을 저장한 뒤 Claude Desktop을 다시 실행합니다.  
다시 실행해야 새 MCP 서버 설정이 반영되는 경우가 많습니다.

## Cursor에서 등록하기

Cursor를 사용하는 경우에는 설정 화면의 MCP 항목에서 서버를 등록합니다.  
서버 이름은 `wikidocs`처럼 알아보기 쉬운 이름으로 정합니다.

```json
{
  "mcpServers": {
    "wikidocs": {
      "url": "https://wikidocs.net/mcp",
      "headers": {
        "Authorization": "Token 발급받은토큰"
      }
    }
  }
}
```

> 이미지 캡처 위치: Cursor 설정 화면에서 MCP 서버를 추가한 화면

등록 후에는 Cursor에서 사용할 수 있는 MCP 도구 목록에 위키독스 관련 도구가 나타나는지 확인합니다.

## 등록 후 확인하기

도구 등록이 끝났다면 가장 먼저 책 목록을 조회해 봅니다.

```text
내 위키독스 책 목록을 보여 주세요.
```

책 목록이 보이면 연결이 정상입니다.  
목록이 보이지 않거나 인증 오류가 나오면 토큰, 헤더 이름, 서버 URL, 도구 재시작 여부를 차례대로 확인합니다.

MCP 설정은 도구별 버전에 따라 화면이나 파일 위치가 달라질 수 있습니다.  
하지만 핵심은 항상 같습니다. `https://wikidocs.net/mcp` 서버에 `Authorization: Token ...` 형식의 인증 정보를 함께 전달해야 합니다.
