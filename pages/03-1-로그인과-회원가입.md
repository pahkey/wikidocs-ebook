위키독스 웹 편집기로 책을 작성하려면 먼저 위키독스에 로그인해야 합니다.  
계정이 없다면 회원가입을 먼저 진행합니다.

## 로그인하기

위키독스 홈페이지에 접속한 뒤 화면 오른쪽 위의 로그인 링크를 선택합니다.

- [https://wikidocs.net](https://wikidocs.net)

> 이미지 캡처 위치: 위키독스 홈페이지 오른쪽 위의 로그인 링크가 보이는 화면

![](../assets/Pasted%20image%2020260426120847.png)

위키독스는 여러 로그인 방법을 제공합니다.  
구글 계정, 페이스북 계정, 또는 위키독스 자체 계정으로 로그인할 수 있습니다.

> 이미지 캡처 위치: 위키독스 로그인 화면의 로그인 방법 선택 영역

![](../assets/Pasted%20image%2020260426121014.png)

이미 계정이 있다면 앞으로 책을 관리할 때도 같은 계정으로 로그인하는 것이 좋습니다.  
책의 소유자와 관리 권한이 로그인 계정에 연결되기 때문입니다.

## 회원가입하기

위키독스 자체 계정을 만들려면 로그인 화면에서 회원가입 링크를 선택합니다.  
회원가입 화면에서는 이메일, 이름, 비밀번호를 입력합니다.

> 이미지 캡처 위치: 위키독스 회원가입 화면의 입력 항목

![](../assets/Pasted%20image%2020260426121100.png)

이메일 주소는 나중에 비밀번호를 찾거나 계정 관련 안내를 받을 때 사용됩니다.  
실제로 확인할 수 있고 오래 사용할 수 있는 이메일을 입력하는 것이 좋습니다.

## 계정 선택 시 확인할 것

위키독스는 이메일 주소를 기준으로 계정을 관리합니다.  
같은 이메일을 사용하면 로그인 방식이 달라도 같은 계정으로 인식될 수 있습니다.

따라서 구글이나 페이스북 같은 외부 계정으로 로그인할 때도 어떤 이메일을 사용하는지 확인해 두는 것이 좋습니다.  
책을 만든 뒤에는 같은 계정으로 로그인해야 책을 계속 관리할 수 있습니다.
