MCP를 사용하면 외부 도구나 LLM이 위키독스 책과 직접 상호작용할 수 있습니다.  
페이지 목록을 확인하고, 원고를 읽고, 필요한 내용을 작성하거나 수정하는 흐름을 만들 수 있습니다.

## MCP 방식이란

MCP는 도구와 서비스가 서로 연결될 수 있도록 해 주는 방식입니다.  
위키독스 MCP를 사용하면 LLM 도구가 위키독스 책의 페이지를 읽고 수정하는 작업을 도울 수 있습니다.

예를 들어 특정 페이지의 내용을 불러와 문장을 다듬거나, 새 페이지의 초안을 만들거나, 여러 페이지의 표현을 일정하게 정리하는 작업에 활용할 수 있습니다.

## 이런 경우에 어울립니다

MCP 방식은 LLM을 적극적으로 활용하려는 저자에게 잘 맞습니다.  
단순히 문장을 복사해 붙여 넣는 대신, 도구가 책의 구조를 이해하고 필요한 작업을 수행하도록 만들 수 있습니다.

반복되는 수정 작업이나 여러 페이지를 함께 살펴봐야 하는 작업에도 유용합니다.  
다만 원고를 자동으로 수정할 수 있는 만큼, 반영 전에는 변경 내용을 직접 확인하는 습관이 필요합니다.

## 진행 흐름

MCP 방식은 보통 다음 흐름으로 사용합니다.

1. MCP를 지원하는 LLM 도구를 준비합니다.
2. 위키독스와 연결할 MCP 설정을 등록합니다.
3. 사용할 계정이나 인증 정보를 설정합니다.
4. LLM 도구에서 책 목록이나 페이지 내용을 불러옵니다.
5. 초안 작성, 문장 수정, 구조 점검 같은 작업을 요청합니다.
6. 변경 내용을 확인한 뒤 필요한 부분만 반영합니다.
