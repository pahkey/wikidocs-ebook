위키독스 옵시디언 플러그인을 사용하려면 먼저 옵시디언에 플러그인을 설치하고, 위키독스 계정과 연결해야 합니다. 이 과정이 끝나면 옵시디언에서 위키독스 책을 불러와 편집할 수 있습니다.

## 커뮤니티 플러그인 사용하기

옵시디언 설정을 열고 `커뮤니티 플러그인` 메뉴로 이동합니다. 처음 커뮤니티 플러그인을 사용하는 경우에는 제한 모드를 해제하고 커뮤니티 플러그인을 사용할 수 있도록 켭니다.

![](../assets/Pasted%20image%2020260429095926.png)

커뮤니티 플러그인은 옵시디언 기본 기능이 아니라 사용자가 추가로 설치하는 기능입니다. 플러그인을 설치할 때는 이름, 설명, 제작자 정보를 확인하고 필요한 플러그인만 설치하는 것이 좋습니다.

## WikiDocs 플러그인 설치하기

커뮤니티 플러그인 화면에서 `탐색` 버튼을 선택합니다. 검색창에 `WikiDocs`를 입력한 뒤 검색 결과에서 WikiDocs 플러그인을 선택합니다.

![](../assets/Pasted%20image%2020260429100214.png)

플러그인 상세 화면에서 설치를 진행하고, 설치가 끝나면 플러그인을 활성화합니다. 활성화까지 마쳐야 옵시디언 왼쪽 영역에 위키독스 관련 아이콘이 표시됩니다.

![](../assets/Pasted%20image%2020260429100304.png)

## API 토큰 준비하기

플러그인이 위키독스 책에 접근하려면 API 토큰이 필요합니다. API 토큰은 위키독스 계정과 플러그인을 연결하는 인증 정보입니다.

위키독스에 로그인한 뒤 `계정설정`으로 이동합니다. 그다음 `API 토큰` 메뉴를 열고 토큰 보기 버튼을 선택하여 API 토큰을 복사합니다.

![](../assets/Pasted%20image%2020260429100443.png)

API 토큰은 비밀번호처럼 다루어야 합니다. 다른 사람에게 공유하지 말고, 화면 캡처를 찍을 때도 토큰 값이 그대로 노출되지 않도록 주의합니다.

## 플러그인 설정 입력하기

다시 옵시디언으로 돌아와 WikiDocs 플러그인 설정 화면을 엽니다. 설정 화면에는 API 주소와 API 토큰을 입력하는 항목이 있습니다.

API 주소에는 다음 값을 입력합니다.

```text
https://wikidocs.net/napi
```

API 토큰 항목에는 앞에서 복사한 위키독스 API 토큰을 붙여 넣습니다. 입력을 마치면 설정을 저장하거나 화면을 닫아도 됩니다.

![](../assets/Pasted%20image%2020260429100539.png)

## 설치 확인하기

설치와 설정이 끝나면 옵시디언 왼쪽에 위키독스 관련 아이콘이 표시됩니다. 책 모양 아이콘은 위키독스 책을 관리할 때 사용합니다.

![](../assets/Pasted%20image%2020260429111335.png)

아이콘이 보이고 책 목록을 불러올 수 있다면 기본 설치는 끝난 것입니다.
