## 1. 위키독스에서 깃허브 연동 책 만들기

위키독스에서는 일반 책과 `깃허브 연동 책`을 만들 수 있습니다. 이 책에서 설명하는 방식은 깃허브 연동 책을 기준으로 하므로, 처음부터 일반 책이 아닌 깃허브 연동 책으로 시작해야 합니다.

test

## 2. 생성된 저장소를 로컬 PC로 가져오기

깃허브 연동 책을 만들면 연결된 깃허브(GitHub) 저장소가 생성됩니다. 깃허브에 접속해 해당 저장소 주소를 확인한 뒤, 로컬 PC로 내려받아 작업을 시작합니다.

## 3. 옵시디언에서 문서 편집하기

저장소를 특정 폴더로 내려받았다면, 해당 폴더를 옵시디언(Obsidian)에서 열어 편집할 수 있습니다. 이 단계에서는 옵시디언에서 목차를 정리하고 문서 내용을 작성하거나 수정합니다.

## 4. LLM을 사용하여 문서 다듬기

옵시디언에서 문서를 편집하는 과정에서 Claude Code, Codex 같은 LLM 도구를 활용해 문서를 다듬을 수 있습니다. 초안을 만들거나 문장을 더 자연스럽게 고치는 데 도움이 됩니다.

## 5. 깃허브에 변경된 내용 반영하기

옵시디언에서 편집한 내용을 깃허브(GitHub)에 커밋하고 push하면 위키독스에 반영됩니다. 깃허브 작업은 터미널에서 직접 수행해도 되고, LLM의 도움을 받아 진행해도 됩니다.
