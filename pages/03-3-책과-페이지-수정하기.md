새 책을 만든 뒤에는 책 전체 설정을 수정하는 화면과 개별 페이지를 수정하는 화면을 구분해서 사용합니다.  
책의 제목이나 공개 여부는 책 수정에서 관리하고, 실제 원고 내용은 페이지 수정에서 관리합니다.

## 책 수정과 페이지 수정의 차이

책 수정은 책 전체에 적용되는 설정을 관리하는 화면입니다.  
책 제목, 책 이미지, 공개 여부, 책 설명, 저작권, 광고 설정처럼 책 전체에 영향을 주는 항목을 수정합니다.

페이지 수정은 각 장의 제목과 내용을 관리하는 화면입니다.  
페이지 제목, 부모 페이지, 공개 여부, 이미지, 본문 내용을 수정할 때 사용합니다.

## 수정 화면으로 들어가기

책이나 페이지를 수정하려면 먼저 독자가 보는 읽기 화면에서 시작합니다.  
위키독스에 로그인한 상태로 내가 관리할 수 있는 책이나 페이지를 열면, 화면 오른쪽 위에 수정 화면으로 이동하는 버튼이 표시됩니다.

책 전체 설정을 바꾸려면 책 읽기 화면에서 오른쪽 위의 수정 버튼을 선택합니다.  
이 버튼을 누르면 책 수정 화면으로 이동합니다.

![](../assets/Pasted%20image%2020260429094828.png)

페이지 내용을 바꾸려면 수정할 페이지의 읽기 화면을 먼저 엽니다.  
그다음 화면 오른쪽 위의 수정 버튼을 선택하면 페이지 수정 화면으로 이동합니다.

![](../assets/Pasted%20image%2020260429094846.png)

수정 화면에 들어간 뒤에는 왼쪽 목록에서 책 제목이나 개별 페이지를 선택해 수정 대상을 바꿀 수도 있습니다.  
책 제목을 선택하면 책 수정 화면으로 이동하고, 개별 페이지를 선택하면 페이지 수정 화면으로 이동합니다.

![](../assets/Pasted%20image%2020260429093850.png)

## 책 수정에서 관리하는 항목

책 수정 화면에서는 다음과 같은 항목을 관리합니다.

![](../assets/Pasted%20image%2020260429093936.png)

- 책 제목
- 책 이미지
- 공개 또는 비공개 여부
- 책 설명 (책 요약)
- 저작권 정보
- 공동 저자
- 광고 표시 여부
- 사용자 스타일/스크립트
- 댓글 알림, 목차 폴딩 같은 기타 설정
- 책 백업과 책 삭제
- 전자책 신청 및 개정

책 제목과 설명(요약)은 독자가 책을 처음 볼 때 확인하는 중요한 정보입니다.  
책의 주제와 대상 독자가 잘 드러나도록 작성하는 것이 좋습니다.

저작권, 광고, 백업, 삭제 같은 항목은 책 운영과 관련된 설정입니다.  
특히 책 삭제는 되돌리기 어렵기 때문에 실행하기 전에 반드시 백업 여부를 확인해야 합니다.

## 페이지와 목차 수정하기

페이지 수정 화면에서는 실제 원고와 목차 구조를 함께 관리합니다.  
페이지 제목, 부모 페이지, 공개 여부, 이미지, 본문 내용을 확인하고 필요한 부분을 수정합니다.

![](../assets/Pasted%20image%2020260429094221.png)

페이지 제목은 목차에 보이는 이름입니다.  
목차 순서를 일정하게 보이게 하려면 제목 앞에 `01`, `02`, `03-1`처럼 번호를 붙이는 방식이 도움이 됩니다.

페이지의 상위 장과 하위 장 관계는 `부모 페이지` 항목에서 설정합니다.  
부모 페이지를 지정하면 해당 페이지가 그 페이지의 하위 장으로 들어가고, 부모 페이지를 지정하지 않으면 최상위 장으로 표시됩니다.

수정 화면의 왼쪽에는 현재 책의 목차가 표시됩니다.  
이 목차를 보면서 어떤 페이지가 있는지 확인하고, 수정할 페이지로 이동할 수 있습니다.

![](../assets/Pasted%20image%2020260429145719.png)

새 페이지를 만들 때는 왼쪽 목차 영역의 `신규 페이지 추가` 기능을 사용합니다.  
이 기능으로 페이지를 추가한 뒤 페이지 제목과 부모 페이지를 정하고 본문을 입력하면 책에 새로운 페이지가 생깁니다.

## 저장, 복제, 하위 페이지, 삭제

페이지 수정 화면 아래에는 페이지를 관리하는 버튼이 있습니다.  
저장은 현재 페이지의 변경 내용을 반영하고, 복제는 현재 페이지를 복사해 새 페이지를 만듭니다.

![](../assets/Pasted%20image%2020260429094331.png)

하위 페이지 추가 기능을 사용하면 현재 페이지 아래에 새 하위 장을 만들 수 있습니다.  
삭제는 현재 페이지를 제거하는 기능이므로 실행하기 전에 정말 삭제해도 되는지 확인해야 합니다.
