# 위키독스로 책 쓰기

책을 쓰고 싶다는 생각은 있어도 막상 시작하려고 하면 막막합니다.  
원고를 어디에 보관해야 할지, 어떻게 공개해야 할지, 나중에 고치고 관리하는 일은 어떻게 해야 할지 고민이 이어집니다.

위키독스는 이런 부담을 줄여 주는 좋은 출발점입니다.  
처음부터 완벽한 책을 완성하지 않아도 됩니다. 목차를 만들고, 한 페이지씩 쓰고, 독자의 반응을 보며 조금씩 고쳐 나가면 됩니다.

이 책은 위키독스에 책을 만들고 싶은 사람을 위한 실전 안내서입니다.  
내가 알고 있는 지식, 경험, 공부한 내용을 온라인 책으로 정리하고 공개하는 과정을 차근차근 따라가 봅니다.

이 책에서는 웹 편집기, 옵시디언 플러그인, MCP, 깃허브 연동 방식까지 위키독스에서 책을 작성하는 여러 방법을 차례로 살펴봅니다.  
아이디어를 초안으로 바꾸고, 초안을 읽기 좋은 글로 고치고, 자신에게 맞는 방식으로 책을 계속 발전시키는 흐름을 익힐 수 있습니다.

책을 쓰는 일은 거창한 출판 프로젝트가 아니라, 내가 알고 있는 것을 정리하고 나누는 과정이 될 수 있습니다.  
이 책을 따라가다 보면 위키독스에 나만의 책을 만들고, 공개하고, 꾸준히 고쳐 나가는 기본 흐름을 갖추게 됩니다.

이 책은 이런 분에게 어울립니다.

- 공부한 내용을 책으로 정리해 보고 싶은 분
- 흩어진 글을 체계적인 책으로 정리해 보고 싶은 분
- LLM을 활용해 원고 작성과 수정의 부담을 줄이고 싶은 분
- 위키독스에 책을 만들고 싶지만 어디서부터 시작해야 할지 막막한 분
